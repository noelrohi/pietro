# Pietro

A workout tracking app for iOS built with SwiftUI and SwiftData.

## Features

- **Workout Library** - Create and manage custom workouts with exercises
- **Exercise Tracking** - Track sets, reps, and weights for each exercise
- **Categories** - Organize workouts by Push, Pull, and Core categories
- **Progress Tracking** - View workout history and track your fitness journey
- **Streak Tracking** - Stay motivated with workout streak statistics

## Tech Stack

- SwiftUI
- SwiftData
- iOS 17+

## Architecture

- **Models/** - SwiftData models for Workout, Exercise, and CompletedWorkout
- **Views/** - Screen-level views (Today, Library, Progress)
- **Components/** - Reusable UI components
- **Theme/** - Custom color theme and styling
