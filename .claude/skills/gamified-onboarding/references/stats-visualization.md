## Animation patterns

### Reveal sequence

1. Card fades in (0.2s)
2. Label appears (0.1s)
3. Number counts up (0.5s)
4. Bar fills from left (0.5s)

### Comparison transition

1. Show current stats (hold 2s)
2. Bars pulse/glow
3. Transition wipe
4. Show potential stats with celebration

### Level up animation

1. Bar fills to 100%
2. Flash/glow effect
3. Number increments
4. New bar starts at 0%

## Color system

```
Very low (0-20):   #ef4444 (red)
Low (20-40):       #f97316 (orange)
Medium (40-60):    #eab308 (yellow)
Good (60-80):      #22c55e (green)
Excellent (80-100): #06b6d4 (cyan/teal)
```

## Typography

- **Stat label:** Medium weight, secondary color
- **Stat number:** Bold, large, primary/accent color
- **Subtext:** Small, muted color

## Pitfalls

- Bars that don't animate (static feels dead)
- Numbers without context (what's good?)
- Missing the before/after comparison
- Stats that don't map to quiz answers
- Color coding that's unclear
- Too many stats at once (4-6 max per screen)
